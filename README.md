

# CSCE 864: Internet Systems and Programming - Spring 2024

A collection of web development projects exploring HTML, CSS, JavaScript, and PHP fundamentals.

## Projects

### 1. Number Guessing Game
A browser-based game where players guess a random number between 1 and 100.
- 10 attempts to guess the correct number
- Provides "too high" or "too low" hints
- Tracks previous guesses to prevent duplicates
- Built with HTML, CSS, and vanilla JavaScript

**File:** `number_guessing_game.html`

### 2. Whac-a-Mole Game
A classic arcade-style game implementation with adjustable difficulty.
- 3x3 grid gameplay
- 60-second timer
- Two difficulty modes: Simple (1s interval) and Hard (0.5s interval)
- Score tracking
- Built with HTML, CSS, and JavaScript

**Directory:** `whac-a-mole game/`

### 3. Chessboard Generator
A PHP-generated chessboard pattern display.

**Directory:** `chessboard/`

### 4. PHP Exercises
Various PHP exercises demonstrating server-side programming:
- **Feedback Form** - AJAX-based form submission with message counter
- **Time Display** - Server-side time rendering
- **Chessboard** - PHP-generated chessboard pattern

**Directory:** `php/`

### 5. Quizzes
In-class quizzes and assignments covering web development concepts.

**Directory:** `quizes/`

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- PHP
- AJAX/XMLHttpRequest

## How to Run
1. For HTML/JS projects: Open the `.html` files directly in a web browser
2. For PHP projects: Run on a local server (e.g., XAMPP, MAMP, or `php -S localhost:8000`)
